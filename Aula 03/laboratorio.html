Resultados da Pesquisa Avançada (5 Tags Extras)

1. Tag: <code>

Função: Define um fragmento de código de computador. Renderiza o texto em uma fonte monoespaçada por padrão.

Exemplo no Código: <code>console.log("Olá Mundo");</code>

Resultado Esperado:

O comando para exibir mensagens no terminal é console.log("Olá Mundo");.

2. Tag: <abbr>

Função: Define uma abreviação ou sigla. Ao passar o mouse sobre o elemento, o atributo title exibe a descrição completa.

Exemplo no Código: <abbr title="HyperText Markup Language">HTML</abbr>

Resultado Esperado:

Estruturamos toda a feira usando apenas HTML. (Passe o mouse por cima da sigla para testar).

3. Tag: <progress>

Função: Exibe nativamente uma barra de progresso indicando a conclusão de uma tarefa.

Exemplo no Código: <progress value="75" max="100"></progress>

Resultado Esperado:

Meta de arrecadação de alimentos da feira atingida em: 75% (75%)

4. Tag: <kbd>

Função: Representa uma entrada de teclado inserida pelo usuário. Também utiliza fonte monoespaçada.

Exemplo no Código: <kbd>Ctrl</kbd> + <kbd>F5</kbd>

Resultado Esperado:

Para atualizar a página limpando o cache do navegador, pressione Ctrl + F5.

5. Tag: <del> e <ins> (Tratadas juntas como um sistema de revisão)

Função: <del> define um texto que foi deletado do documento (renderiza riscado). <ins> define o texto que foi inserido no lugar (renderiza sublinhado).

Exemplo no Código: <del>R$ 150,00</del> <ins>R$ 90,00</ins>

Resultado Esperado:

O preço do ingresso promocional mudou de R$ 150,00 para R$ 90,00 hoje!
